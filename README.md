# ManagementGameQT

Учебный проект на C++ с графическим интерфейсом, реализованный с использованием Qt Widgets.  
Проект предназначен для демонстрации базовых навыков разработки desktop-приложений на C++ и работы с Qt.

## Технологии
- C++17
- Qt Widgets
- Qt Network
- CMake
- Git

## Возможности
- Графический пользовательский интерфейс
- Игровая логика и обработка событий
- Модульная структура проекта
- Сборка проекта с использованием CMake

## Сборка и запуск

### Требования
- Компилятор с поддержкой C++17
- Qt (с поддержкой Qt Widgets)
- CMake 3.10+

### Сборка
- git clone https://github.com/7malahit7/ManagmentGameQT.git
- cd ManagmentGameQT
- mkdir build && cd build
- cmake ..
- cmake --build .

## Запуск выполняется из каталога build после успешной компиляции.

## Структура проекта
- src — исходный код
- include — заголовочные файлы
- CMakeLists.txt — конфигурация сборки
